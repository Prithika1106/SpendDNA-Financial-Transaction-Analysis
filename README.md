# 💳 SpendDNA – Financial Transaction Analysis

## 📌 Overview

SpendDNA is a fintech-inspired data analytics project developed using **Python, Pandas, and NumPy**. The project transforms raw financial transaction data into meaningful insights by cleaning the data, extracting vendors, categorizing expenses, detecting anomalies, analyzing spending behavior, and identifying financial archetypes.

This project demonstrates practical data analytics techniques commonly used in banking and fintech applications.

---

## 🚀 Features

* Transaction data cleaning and preprocessing
* Multi-format date and amount parsing
* Vendor extraction and standardization
* Expense categorization
* Spending overview dashboard
* Monthly spending trend analysis
* Time-of-day spending analysis
* Z-score based anomaly detection
* Spending archetype detection
* Executive financial summary

## ▶️ How to Run

1. Clone this repository.
2. Install the required libraries.

```bash
pip install -r requirements.txt
```

3. Place the transaction dataset in the project folder.
4. Open the notebook in Jupyter Notebook or Google Colab.
5. Run all cells from top to bottom.
